
EduHub - Anurag University (Creative Student Portal)
Static website prepared for academic assignment.

How to preview:
- Unzip and open index.html in browser (offline).
- Or use VS Code Live Server for local preview.

Files:
- 10 HTML pages: index, about, departments, courses, admission, login, register, gallery, contact, feedback
- css/style.css, js/main.js, sample svgs in images/

Note:
All forms are demo-only and show alert on submit. For real deployment, integrate backend.
